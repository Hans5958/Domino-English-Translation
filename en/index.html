"Domino" MIDI/Music Editor Software User Manual

This is the manual for Domino. It is preferred to read them in order from the top.
Since this manual explains how to use the software, the specifications of MIDI and how to compose are not written. You may check other websites about that.

It is difficult to explain all the functions, so the manual have been broken down between multiple parts, but if you have any questions, please ask on the bulletin board, etc.
